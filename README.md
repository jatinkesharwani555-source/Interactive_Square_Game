#  Interactive Square Game

A simple and fun interactive web project built using **HTML, CSS, and JavaScript**.
This project demonstrates **DOM manipulation**, **event handling**, and **dynamic styling**.


##  Features

*  Random number generation on hover
*  Color changing squares
*  Random RGB color generator
*  Interactive UI using mouse events
*  Real-time DOM updates


##  Concepts Used

* DOM Selection (`querySelector`)
* Event Listeners (`mouseover`, `mouseenter`, `mouseleave`)
* Random number generation
* Dynamic CSS manipulation
* JavaScript logic building


##  How It Works

###  Square 1

* On hover → Generates a random number
* On leave → Resets to 1

###  Square 2

* Changes color sequentially:

  * Red → Green → Blue
* On leave → Turns magenta

###  Square 3

* On hover → Generates random RGB color
* On leave → Turns magenta

###  Square 4

* Controls other squares:

  * Changes color of Square 1, 2, and 3 randomly
* On leave → Resets all to magenta


##  Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
